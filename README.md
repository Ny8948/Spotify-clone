# 🎵 Spotify Clone

A responsive Spotify Web Player Clone built using HTML and CSS. This project recreates the Spotify web interface, including the sidebar, navigation bar, music cards, and music player. :contentReference[oaicite:0]{index=0}

---

## 📌 Project Overview

This project is a front-end clone of Spotify designed using only HTML and CSS. It demonstrates responsive layout design, Flexbox, modern UI components, and clean styling. :contentReference[oaicite:1]{index=1}

---

## ✨ Features

- 🎵 Spotify-inspired user interface
- 🏠 Sidebar navigation (Home, Search, Library)
- 📚 Playlist and Podcast sections
- 🎧 Recently Played section
- 🔥 Trending Music cards
- ⭐ Featured Music cards
- 📌 Sticky navigation bar
- 🎼 Bottom music player UI
- 📱 Responsive design
- 🎨 Font Awesome icons
- 🔤 Google Fonts (Montserrat)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/nitin8948/spotify-clone.git
```

2. Open the project folder

```bash
cd spotify-clone
```

3. Open `index.html` in your browser.
